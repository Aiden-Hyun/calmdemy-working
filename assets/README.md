# CalmNest Assets

Place the following image files in this directory:

- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen image (1284x2778 for iPhone, will scale)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)
- `favicon.png` - Web favicon (32x32 or 48x48)

## Design Guidelines

CalmNest uses a calm, peaceful color palette:
- Primary: #6c5ce7 (purple)
- Background: #1a1a2e (dark)
- Accent: #74b9ff (light blue)

Consider using:
- A lotus flower, moon, or zen circle as the icon motif
- Soft gradients in the splash screen
- Minimalist, clean design
